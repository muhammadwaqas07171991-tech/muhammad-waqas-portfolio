Agentic System

K-WaterGuard AI Agent

K-WaterGuard AI is a Python-based agentic water-quality monitoring system for South Korea. It collects public water-quality API data, stores historical records, creates plots and maps, generates an HTML dashboard, and supports optional chatbot integration through a secure backend.
