# Free Portfolio Website Hosting

This portfolio is a static website, so it can be hosted for free.

## Option 1: GitHub Pages

1. Create a GitHub repository, for example `muhammad-waqas-portfolio`.
2. Upload the files from this `portfolio_site` folder.
3. Open the repository settings.
4. Go to `Pages`.
5. Choose the main branch and root folder.
6. Save. GitHub will give you a free public website link.

## Option 2: Netlify

1. Go to Netlify and create a free account.
2. Drag and drop this `portfolio_site` folder into Netlify.
3. Netlify will publish the site and give you a free link.

## Option 3: Vercel

1. Create a free Vercel account.
2. Import a GitHub repository containing these files.
3. Deploy with the default static-site settings.

## Before Publishing

- Upload all `.html`, `.css`, and `.js` files, not only `index.html`.
- Upload the full `publications` folder with the site files.
- Upload `Logo.png` and `753134.png` with the site files.
- Replace or remove any contact information you do not want public.
- Check every publication title and year.
- Confirm that the downloadable CV is the version you want visitors to see.
- Consider exporting your CV as PDF later, because PDF is easier for visitors to open than DOCX.
